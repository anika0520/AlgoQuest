# ⚔ AlgoQuest – Interactive Algorithm Playground

**AlgoQuest** is an interactive web-based platform designed to help students and developers learn algorithms through visualizations, simulations, and mini-games.

Instead of reading static code, users can **see algorithms execute step-by-step** and understand their behavior visually.

The platform combines **algorithm visualizers, dynamic programming tables, graph simulators, complexity charts, and gamified learning** to make algorithm learning engaging and intuitive.

---

# 🚀 Features

## 📊 Sorting Algorithm Visualizer

Algorithms included:

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
* Heap Sort
* Counting Sort
* Radix Sort

Features:

* Animated bar visualization
* Adjustable speed
* Step-by-step execution
* Complexity analysis
* Code snippets (Python / Java / C++)

---

## 🕸 Graph Algorithm Visualizer

Interactive graph builder where users can create nodes and edges.

Algorithms included:

* Breadth First Search (BFS)
* Depth First Search (DFS)
* Dijkstra’s Algorithm
* Prim’s Minimum Spanning Tree
* Kruskal’s Algorithm
* Bellman-Ford
* Topological Sorting

Features:

* Click-based node creation
* Traversal animation
* Algorithm logs
* Complexity display

---

## 🧮 Dynamic Programming Visualizer

Understand how DP tables are filled step by step.

Algorithms included:

* 0/1 Knapsack
* Longest Common Subsequence (LCS)
* Matrix Chain Multiplication
* Coin Change

Features:

* State table visualization
* Subproblem tracking
* Optimal solution highlighting

---

## 💡 Greedy Algorithms

Visual demonstrations of greedy strategies such as:

* Huffman Coding
* Fractional Knapsack
* Activity Selection

---

## 📝 String Algorithms

Pattern matching and string data structure visualization.

Includes:

* KMP Pattern Matching
* Rabin-Karp Algorithm
* Trie Data Structure

---

## 🎮 Algorithm Mini-Games

Gamified learning experience with challenges such as:

* Sorting race
* DFS maze solver
* Graph path puzzles
* Huffman encoding game

Players earn **XP rewards and badges**.

---

## 📈 Time Complexity Visualizer

Interactive graphs to understand algorithm growth rates:

* O(1)
* O(log n)
* O(n)
* O(n log n)
* O(n²)

Helps compare algorithm efficiency visually.

---

## 🏆 Leaderboard System

Track your progress and compete with others.

Features:

* XP based ranking
* Achievement badges
* Algorithm completion rewards
